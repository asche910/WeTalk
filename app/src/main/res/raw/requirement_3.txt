    <p><strong>一、项目背景：</strong></p>

<p>&nbsp; &nbsp; &nbsp; &nbsp;寻找可以应用在洗衣机上，杀死水中细菌的新技术，使洗衣机在洗涤过程中不仅能清除污垢，也能起到一定的杀菌效果，并效果可以呈现（用户能够感受到）</p>

<p><strong>二、需求参数：</strong></p>

<p>&nbsp; &nbsp; &nbsp; &nbsp;对杀菌模块：要求模块尺寸能够安装在洗衣机上，并且使用寿命大于10年。</p>

<p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;备注：模块尺寸视安装位置而定，可以在实际方案中沟通协商确定</p>

<p>&nbsp; &nbsp; &nbsp; &nbsp;对杀菌材料：要求材料不易脱落，加工工艺可行，并且使用寿命大于10年。</p>

<p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;备注：具体工艺要求，视方案类型，共同协商确定</p>

<p><strong>三、不感兴趣的方向：</strong></p>

<p>&nbsp; &nbsp; &nbsp; &nbsp;臭氧、紫外、纳米银离子等方式，不在本需求考虑范围内</p>
